# 🛒 SmartCart - Flutter E-commerce App

SmartCart is a modern, scalable e-commerce mobile application built using **Flutter** with a **Firebase backend**.
It features **AI-powered product search**, **secure payments via Razorpay**, and a clean **MVVM architecture**.

---

## 🚀 Features

* 🔐 **User Authentication**

  * Firebase Email/Password login & signup

* 🛍 **Product Listing**

  * Real-time data from Firestore
  * Category-based filtering

* 🤖 **AI-Powered Search**

  * Integrated with OpenAI API
  * Smart product suggestions

* ❤️ **Wishlist & Cart**

  * Add/remove items
  * Real-time sync with Firebase

* 💳 **Secure Payments**

  * Razorpay integration
  * Payment success & failure handling

* 📦 **Order Management**

  * Order history
  * Real-time updates

* 🌙 **Dark/Light Theme**

  * Toggle theme using Provider

* ⚡ **Modern UI/UX**

  * Shimmer loading
  * Responsive design
  * Smooth animations

---

## 🧱 Architecture

This project follows **MVVM (Model-View-ViewModel)** architecture:

* **Model** → Data structure (Product, User, Order)
* **View** → UI Screens
* **ViewModel** → Business logic & state management

---

## 🛠 Tech Stack

* **Frontend**: Flutter (Dart)
* **State Management**: Provider
* **Backend**: Firebase

  * Authentication
  * Firestore Database
  * Storage
* **Payments**: Razorpay
* **AI Integration**: OpenAI API
